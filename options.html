security 禁用搜索建议

这个扩展程序可以帮助您隐藏Chrome地址栏的搜索建议，并清理非白名单网站的历史记录，提高您的浏览隐私。

info 功能介绍

  • 自动隐藏地址栏搜索建议
  • 支持白名单功能，允许特定网站不被清理历史记录
  • 可以通过扩展图标快速将当前网站添加到白名单
  • 在切换标签页时自动清理非白名单网站的历史记录

help_outline 使用方式

  1. 扩展程序安装后会自动隐藏所有网站的地址栏搜索建议
  2. 如果您希望某些网站的历史记录不被清理，可以将其添加到下方的白名单中
  3. 您也可以通过点击扩展图标，快速将当前网站添加到白名单
  4. 每次切换标签页时，扩展程序会自动清理非白名单网站的历史记录
  5. 白名单支持正则表达式匹配，例如 ".*\\.example\\.com" 可以匹配所有 example.com 的子域名
